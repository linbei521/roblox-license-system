# Roblox云端卡密系统

一个安全、可靠的Roblox付费脚本云端卡密验证系统。

## 功能特性

- 🔐 安全的卡密生成和验证
- 🖥️ Web管理面板
- 🛡️ 防破解和反调试
- 📊 实时数据统计
- 🔄 自动心跳检测
- 📱 移动端友好界面

## 技术栈

- **后端**: Node.js + TypeScript + Express + Prisma + PostgreSQL + Redis
- **前端**: React + TypeScript + Ant Design + Vite
- **部署**: Docker + Nginx
- **客户端**: Lua (Roblox)
